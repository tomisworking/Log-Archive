# Log-Archive

Simple Bash script that archives system logs from a specified directory by compressing them into a timestamped file.

The tool:

- **Accepts a log directory as a command-line argument**  
- **Compresses logs into a .tar.gz archive**
- **Creates a timestamped archive filename** 
- **Stores archives in a dedicated directory**  
- **Logs the archive date and time to a log file**
